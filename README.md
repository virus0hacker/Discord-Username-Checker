Check Discord usernames quickly and accurately with this powerful tool.

![Screenshot](https://raw.githubusercontent.com/virus0Hacker/Discord-Username-Checker/main/Screenshot.png)

## Features ✨ :
- Fast username checking
- Automatic saving of available usernames

## Installation 🚀 :

### Easy Method:
```bash
git clone https://github.com/virus0hacker/discord_checker.py
```
pip install -r requirements.txt

python discord_checker.py

Requirements:
---
Python 3.8+

aiohttp

colorama
